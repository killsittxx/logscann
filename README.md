# Проект: LogScan 🛡️

**LogScan** — это консольная утилита для специалистов по кибербезопасности и системных администраторов. Программа сканирует лог-файлы, находит в них подозрительные IP-адреса и хэши файлов (SHA256) и проверяет их на вредоносность.

## Основные возможности
- **Умный поиск**: Использует регулярные выражения для точного извлечения данных.
- **Интеграция с VirusTotal**: Проверяет объекты через API (поддерживает Mock-режим, если нет ключа).
- **Гибкие отчеты**: Сохраняет результаты в формате JSON или CSV.
- **Логирование**: Все действия и ошибки записываются в файл `app.log`.
- **Модульная структура**: Код разбит на логические блоки для удобства.

##  Структура проекта
- `main.py` — точка входа, управление командами.
- `scanner.py` — поиск IP и хэшей в тексте логов.
- `api_client.py` — общение с внешними сервисами проверки.
- `reporter.py` — создание и запись итоговых отчетов.
- `utils.py` — настройки логирования системы.
- `logs/` — папка с примерами лог-файлов (например, `sample.log`).

- `output/` — папка, куда сохраняются отчеты.
